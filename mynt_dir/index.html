{% extends 'layout.html' %} {% block content %} A scarily powerful programming language for .Net which has:

An ultra clean syntax:

Python-style indentation, automatic type inference and a minimum of brackets and keywords makes for highly readable code. See the difference for yourself with this online converter.

Powerful extension features:

add new keywords, add instructions to modify/analyse your code during compilation or even create your own DSL!

IDE support:

Boo has been supported in Sharpdevelop since 2006. There are also plugins for sublime and Notepad++ plug-in https://github.com/Shammah/boo-sublime https://groups.google.com/forum/#!topic/boolang/WUcSQOn0nEU

First class Functions:

All the speed of C#:

While IronPython is a great project, sometimes you need your application to be "snappy". Boo is statically compiled, just the same as C# or VB.Net, meaning any speed differences should be marginal. You can fo course use assemblies written in Boo by C# and vice-versa. (although there is the option to duck type).

Getting started:

Get yourself a copy of Sharpdevelop 3.2 or 4.3, Boo comes included so you'll be able to produce .Net applications. https://github.com/bamboo/boo/wiki/Boo-Primer https://github.com/bamboo/boo/wiki/Tutorials
